My Gemini Chatbot

A simple AI chatbot web app built with Python, Google Gemini API, and Streamlit

### What it does
- Chat with Google's Gemini AI in real time.
- Remembers conversation history during the session.
- Clean and simple web interface.

### Technologies
- Python
- Google Gemini API (google-genai)
- Streamlit

### How to Run
### 1. Clone the repository
git clone https://github.com/Okparaji Wisdom/gemini-chatbot.git

### 2. Install dependencies
pip install streamlit google-genai

### 3. Add your API Key
Create a .env file and add:
GEMINI_API_KEY=your_api_key_here

### 4. Run the app
streamlit run app.py
